# 小狼毫配置优化计划

## 需求分析
1. 使用 JetBrainsMono Nerd Font 字体
2. 添加 `,` 和 `.` 作为翻页键
3. 增大字体大小

## 实现方案

### 1. 配置 JetBrainsMono Nerd Font 字体
- **文件**: `weasel.custom.yaml`
- **修改**: 在 `patch` 部分更新字体配置，将 JetBrainsMono Nerd Font 添加到字体列表中
- **说明**: 小狼毫支持 Nerd Font，使用后会使界面更加现代化和美观，特别是在显示特殊符号时

### 2. 添加 `,` 和 `.` 翻页键
- **文件**: `default.custom.yaml`
- **修改**: 在 `key_binder/bindings` 中添加翻页键绑定
- **说明**: 通过配置快捷键绑定，实现逗号上翻页、句号下翻页的功能

### 3. 增大字体大小
- **文件**: `weasel.custom.yaml`
- **修改**: 调整 `font_point` 参数，增大全局字体大小
- **说明**: 适当增大字体可以提高可读性

## 具体修改内容

### 1. 更新 `weasel.custom.yaml` 配置字体
```yaml
patch:
  # 字体设置
  "style/font_face": "JetBrainsMono Nerd Font, Segoe UI Emoji:23:23, Segoe UI Emoji:2a:2a, Segoe UI Emoji:30:39, Segoe UI Emoji:fe0f:fe0f, Segoe UI Emoji:20e3:20e3, Microsoft YaHei UI, Segoe UI, DengXian, Segoe UI Emoji, Noto Color Emoji, Segoe UI Symbol"
  "style/font_point": 18  # 增大字体大小
  "style/label_font_point": 16  # 增大标签字体大小
  "style/comment_font_point": 15  # 增大注释字体大小
```

### 2. 更新 `default.custom.yaml` 添加翻页键
```yaml
patch:
  key_binder/bindings/+:
    # 开启逗号句号翻页
    - { when: paging, accept: comma, send: Page_Up }
    - { when: has_menu, accept: period, send: Page_Down }
```

## 预期效果
1. 输入法界面将使用 JetBrainsMono Nerd Font 字体，更加现代化美观
2. 可以使用 `,` 和 `.` 进行翻页，操作更加便捷
3. 字体增大后，可读性将显著提高

## 注意事项
1. 确保 JetBrainsMono Nerd Font 字体已经正确安装在系统中
2. 修改后需要重启小狼毫输入法才能生效
3. 字体大小可以根据个人喜好进一步调整
