# 小鹤双拼初学者提示配置计划

## 需求分析
用户是小鹤双拼初学者，需要配置键位提示和学习辅助功能，以便更好地学习和使用双拼。

## 实现方案

### 1. 增强拼音注释显示
- **文件**: `double_pinyin_flypy.schema.yaml`
- **修改**: 优化 `translator` 部分的 `spelling_hints` 和 `comment_format` 配置
- **说明**: 增强拼音注释的显示，让初学者能清楚看到当前输入的双拼对应的全拼

### 2. 优化预编辑显示
- **文件**: `double_pinyin_flypy.schema.yaml`
- **修改**: 调整 `preedit_format` 配置
- **说明**: 优化输入编码的显示，让初学者能直观看到双拼键位对应的拼音

### 3. 配置学习辅助功能
- **文件**: `double_pinyin_flypy.schema.yaml`
- **修改**: 调整相关过滤器和辅助功能
- **说明**: 启用和优化学习辅助功能，如错音提示、常用词置顶等

### 4. 配置自定义提示样式
- **文件**: `weasel.custom.yaml`
- **修改**: 调整候选窗口的样式和布局
- **说明**: 优化候选窗口的显示效果，让提示信息更加醒目

## 具体修改内容

### 1. 增强拼音注释显示
```yaml
# 在 double_pinyin_flypy.schema.yaml 中修改
patch:
  translator:
    spelling_hints: 8             # 显示拼音提示，数值越大提示越多
    always_show_comments: true    # 始终显示注释，方便查看拼音
    comment_format:               # 优化注释显示格式
      - xform/^/「/               # 使用更醒目的括号
      - xform/$/」/               # 使用更醒目的括号
```

### 2. 优化预编辑显示
```yaml
# 在 double_pinyin_flypy.schema.yaml 中修改
patch:
  preedit_format:               # 优化预编辑显示，让双拼键位对应的拼音更清晰
    - xform/(^|[ '])aa/$1a/     # 保持原有的键位映射规则
    - xform/(^|[ '])ee/$1e/     # 保持原有的键位映射规则
    - xform/(^|[ '])oo/$1o/     # 保持原有的键位映射规则
    # 其他键位映射规则保持不变
```

### 3. 配置学习辅助功能
```yaml
# 在 double_pinyin_flypy.schema.yaml 中修改
patch:
  # 调整过滤器顺序，优化学习辅助功能
  filters:
    - lua_filter@*corrector                         # 错音错字提示，帮助初学者纠正错误
    - lua_filter@*pin_cand_filter                   # 置顶常用词，减少输入负担
    - simplifier@emoji                              # Emoji
    - simplifier@traditionalize                     # 简繁切换
    - lua_filter@*search@radical_pinyin             # 部件拆字辅码
    - uniquifier                                    # 去重
```

### 4. 配置自定义提示样式
```yaml
# 在 weasel.custom.yaml 中修改
patch:
  # 优化候选窗口样式，让提示更醒目
  "style/layout/hilite_padding": 12                # 增加高亮区域内边距
  "style/layout/hilite_spacing": 10               # 增加高亮区域间距
  "style/layout/corner_radius": 10                # 优化圆角
  "style/layout/shadow_radius": 5                 # 增加阴影效果
  
  # 优化配色方案，增强提示对比度
  "preset_color_schemes/modern_dark":
    comment_text_color: 0x6A9955                  # 注释文字颜色，使用醒目的绿色
    hilited_comment_text_color: 0xB5CEA8          # 高亮注释文字颜色，使用更醒目的绿色
```

## 预期效果
1. 拼音注释会更加醒目，初学者能清楚看到当前输入的双拼对应的全拼
2. 预编辑显示优化，让初学者能直观看到双拼键位对应的拼音
3. 学习辅助功能启用，如错音提示、常用词置顶等，帮助初学者更好地学习
4. 候选窗口样式优化，提示信息更加醒目，提高可读性

这些修改组合使用，将为小鹤双拼初学者提供更好的学习体验，帮助他们更快地掌握双拼输入。
